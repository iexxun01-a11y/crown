서비스별 진행 방식

한 페이지 안에서도 어떤 서비스를 요청해야 하는지 바로 구분되도록 핵심 흐름만 정리했습니다.

공항픽업

항공편 확인 → 미팅 위치 안내 → 호텔 또는 카지노 이동

도착 시간과 인원을 기준으로 차량 타입을 정하고, 초행자도 찾기 쉬운 미팅 위치를 안내합니다.

패스트트랙

입출국 조건 확인 → 가능 여부 안내 → 공항 동선 정리

공항 상황과 일정 조건에 따라 대기 시간을 줄이는 방향으로 안내합니다.

차량이동

목적지 확인 → 차량 배정 → 대기·샌딩 연결
